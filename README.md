# Inti

A collection of different applications and procedures used in plant breeding and experimental designs.

## Install

```{r eval=F}
install.packages("devtools")
devtools::install_github("flavjack/inti")
```
## Shiny apps

### Yupana

```{r eval=F}
inti::yupana()
```
### Tarpuy

```{r eval=F}
inti::tarpuy()
```

### Rticles

```{r eval=F}
inti::rticles_addin()
```
